Jest is included in create-react-app and is a great test runner and assertion library for JavaScript applications. It can be used without react and also brings tools like watching for file changes, testing only changed (and depending) tests and integrates code coverage tools

Enzyme

React Testing Library

Enzyme is the most popular renderer and helper

But I like the API of react testing library better. It is shorter and is more user centric than developer centric.

Cypress is my favorite testing tool. You test the rendered web page. Unlike jest which renders into JSDOM, cypress renders in a real browser

You get a nice API to interact with the page. With that you can mimic what a user would be doing.

It also has a nice Test Runner

You can either stub or mock your API calls or let the application call the backend.

it can even run in headles mode in CI

Dashboard paid or free for open source

Visual Regression Testing with percy

When developing your own components it can be great to know that changes you are making won't affect the look of the component.

With visual testing the tool takes a screenshot of your rendered app or component at the time you are satisfied with how it looks and takes another screenshot every time you run the test.

It then compares the screenshots and asks you, if it found changes, whether the changes are wanted or a regression.
