Module bundlers
If you have shared component libraries these tools are great to showcase and document these components.

Storybook

Styleguidist

Docz
  • With all of these you can live edit the code of the components in a playground and for example edit the props.
  • Storybook is the most popular of those. You need to create separate pages and add components and documentation there.
  • Docz is the newcomer. You write your show case pages in MDX. Its Markdown with JSX support
  • My favorite is styleguidist. For every component you write a Readme in markdown. Every code fence you have in there will be rendered. It also loads the JSdoc documentation from your source files.
  • Storybook is prettier and has addon support, docz is easy and pretty. But my main benefit for styleguidist is that it automatically leads to documentation.
