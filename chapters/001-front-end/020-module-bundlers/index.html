When using create react app, you are automatically using Webpack. Configuring Webpack yourself can be quite complicated but it isn't necessary when using CRA.

For component libraries or libraries I recommend rollup. The configuration is straight forward, has good defaults and there are several plugins.

Parcel is another alternative to rollup which claims to have better defaults
