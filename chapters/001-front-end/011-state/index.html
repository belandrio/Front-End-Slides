State Management
Recommended libraries
There are so many ways to handle state in React apps

Hooks

useState
useReducer
They will get you far. They will get you very far. But I think halfway through development you will hit a wtf moment.
Let's say you have a component ChatWindow and a component SingleMessage and they should share a state.
That can get ridicously complex at one point, prop-drilling

Context

The built-in solution is using context. There even is a hook for it.
Context are like a scope. We provide values via a Provider and the whole subtree of components can have access to the values in that scope.
3rd Party state management libraries

Redux

MobX

XState

Redux is like Context paired with useReducer on steroids, it's the most popular 3rd party state manager

It has a great range of available middlewares that maybe help by automatically syncing your state into local storage, or help you with async calls

MobX is the second most popular library. It uses JavaScript decorators and Observables.

XState handles the state inside a state machine.
